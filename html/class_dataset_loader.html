Detailed Description

The DatasetLoader class This class loads the Vespucci Dataset archive (a ZIP file saved as .vds) The constructor initializes the QuaZip object corresponding the the archive. The manifest is loaded using LoadManifest(); The YAML data in the manifest is parsed by yaml-cpp into and the structures are populated by their keys and empty matrices. The empty matrices are then loaded from the .bin files using LoadObjects.
